# Quickstart: Docusaurus Landing Page UI Rebuild

This document outlines the steps to get the new landing page running locally.

## Prerequisites

-   Node.js (v18 or higher)
-   npm or yarn

## Running the development server

1.  **Install dependencies**:
    ```bash
    npm install
    ```
2.  **Start the development server**:
    ```bash
    npm run start
    ```
3.  **Open the site in your browser**:
    [http://localhost:3000](http://localhost:3000)

The new landing page will be visible at the root of the site.
